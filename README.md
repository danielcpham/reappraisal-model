## __Running__: 

To run the algorithm, call the following in terminal while in the repository's main directory:
```shell
python model
```
### Additional Arguments:
- ```-f ``` - Far Away Analysis
- ```-o ``` - Objectivity Analysis
- ```-v ``` : verbose
